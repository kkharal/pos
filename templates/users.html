User Roles Explained

👑 Super Admin

  • Everything Admin can do
  • Create & manage multiple shops
  • Switch between shops (all shops view)
  • See data across all shops
  • System-level configuration
  • Intended for business owners

🏪 Shop Owner

  • Admin-level access for assigned shops
  • Switch between assigned shops only
  • Manage products, stock & settings
  • Create admin & sales staff users
  • View reports for assigned shops
  • Cannot create or delete shops

🛡️ Admin Role

  • Full access to all pages
  • Can see cost prices and profits
  • Manage products and stock
  • View sales history & reports
  • Create and manage users
  • Access dashboard with metrics

💼 User (Sales Staff)

  • Access Point of Sale only
  • Make sales transactions
  • See sell prices only (not cost)
  • Cannot see profit margins
  • Cannot access admin pages
  • Recommended for employees
